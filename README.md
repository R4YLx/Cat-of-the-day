# Cat-of-the-day

[Cat of the day](https://cat4u.netlify.app)

Providing images of random cats with a press of a button.
The images are fetched from [Cataas](https://cataas.com) using async and await.

## Languages and tools
- Vanilla JavaScript
- Bootstrap 5
- VS Code
